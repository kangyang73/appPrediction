### Dependencies

The following modules are needed.

- `Python 2.7`
- `OpenAI gym`
- `Tensorflow`
- `sklearn`
- `numpy`
- `scipy`
- `pandas`

### Setup

```bash
$ cd ./appPrediction/gym
$ pip install -e .
```

### Execution

```bash
$ cd ./appPrediction/src
$ bash run_main.sh
```
